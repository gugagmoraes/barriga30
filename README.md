# Barriga 30 - MVP

Projeto de emagrecimento gamificado com IA.

## Tecnologias

- Next.js 14
- Tailwind CSS
- Supabase
- Stripe
- Shadcn/UI

## Configuração

Para rodar o projeto, é necessário configurar as seguintes variáveis de ambiente:

- `NEXT_PUBLIC_SUPABASE_URL`
- `NEXT_PUBLIC_SUPABASE_ANON_KEY`
- `STRIPE_SECRET_KEY`
- `NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY`

## Deploy

O projeto é implantado automaticamente na Vercel a cada push no branch `main`.
